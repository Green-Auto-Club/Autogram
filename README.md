# Green Auto Club — Adrenalina em Quatro Rodas 🏎️🔥

Bem-vindo ao **Green Auto Club**, a única rede social onde o ego humano é atropelado pela estética mecânica. Se estás à procura de selfies, fotos de pratos gourmet ou frases motivacionais escritas em cima de imagens de pores do sol, vieste ao sítio errado. Aqui, o silício e o aço são os únicos que têm direito a voz.

## 📌 Visão Geral

O **Green Auto Club** é uma plataforma mobile focada exclusivamente no universo automóvel. O conceito é simples, mas ditatorial:
* **Só entram carros.**
* Humanos, animais ou decorações de interiores são considerados "ruído" e eliminados sem piedade pela nossa moderação.

## ⚙️ Arquitetura do Sistema (O Motor)

Para garantir que a plataforma não se desintegre na primeira curva, utilizámos uma stack de alta performance:

| Camada | Tecnologia | Objetivo Operacional |
| :--- | :--- | :--- |
| **Interface** | `React Native` / `TypeScript` | Experiência de utilizador fluida e tipagem forte para evitar que o código "bata" em ambiente mobile. |
| **Backend** | `Node.js` (Microserviços) | Arquitetura distribuída para gerir fluxos de dados massivos sem sobreaquecer o servidor. |
| **Persistência** | `PostgreSQL` | Onde os metadados dos teus ativos são guardados com fidelidade absoluta. |
| **IA / Moderação** | `TensorFlow` / `Rekognition` | O nosso "segurança" digital que higieniza o ecossistema de qualquer presença não-automóvel. |
| **Cloud Infra** | `AWS Ecosystem` | Entrega global de imagens em 4K com latência de um Fórmula 1. |

## 🚀 Funcionalidades Principais

* **Upload de Imagens:** Partilha as tuas máquinas com a comunidade.
* **Sistema de Engagement:** Likes e comentários (desde que sejam sobre o carro).
* **Exploração Especializada:** Filtros por categorias (Supercarros, Clássicos, Tunados, JDM, etc.).
* **Perfis Personalizados:** A tua garagem virtual, onde o teu carro é a foto de perfil.

## 🧠 Moderação Inteligente 

Não confiamos em critérios subjetivos. Utilizamos modelos avançados de visão computacional para garantir a pureza do feed:

1.  **Check de Veículo:** A imagem contém um carro? (Sim -> Procede | Não -> Lixo).
2.  **Bloqueio Automático:** Conteúdo irrelevante (comida, pessoas, paisagens sem estrada) é barrado no upload.
3.  **Revisão de Anomalias:** Conteúdo suspeito é enviado para uma fila de auditoria para garantir que nenhum intruso passa.
